# **App Name**: SimplyStart

## Core Features:

- Note Creation: Allow users to create and save text notes.
- Basic Formatting: Implement simple formatting options like bold, italics, and underline.
- Local Storage: Store notes locally in the browser using local storage.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) for a calming, focused atmosphere.
- Background color: Light, desaturated blue (#F0F4FF).
- Accent color: Light violet (#BDB2FF), for highlights and interactive elements.
- Font: 'Inter', a sans-serif for clear and modern text.
- Clean and minimalist layout to emphasize simplicity and focus on content creation.
